# Network+ N10-009 Flashcards

An interactive CompTIA Network+ N10-009 flashcard study tool built with HTML, CSS, and JavaScript.

## Live Project

This project can be hosted with GitHub Pages by uploading `index.html` to the root of the repository and enabling GitHub Pages from the repository settings.

## What This Covers

- OSI model layers
- TCP vs UDP
- Common ports and protocols
- DHCP, DNS, NAT, and APIPA
- IPv4 addressing and subnetting
- VLANs, trunking, and STP
- Routing concepts
- Wireless networking
- Network security basics
- Troubleshooting tools and commands
- Network+ performance-based question practice concepts

## Why I Built This

I created this as a personal study resource while preparing for the CompTIA Network+ N10-009 certification. The goal is to make review faster, more organized, and easier to repeat before exam day.

## How to Use

1. Open the flashcard page.
2. Choose a domain or study all cards.
3. Click each card to reveal the answer.
4. Use shuffle mode for review.
5. Repeat weak areas until they feel automatic.
